Catalogue TMDB non configuré

Le catalogue cinéma nécessite une clé API The Movie Database.

{{.Error}}

Comment obtenir une clé API

  1. Créez un compte sur themoviedb.org
  2. Allez dans Paramètres → API → Demander une clé API (type « Developer »)
  3. Ajoutez la clé dans votre fichier .env :
TMDB_API_KEY=votre_cle_api

Redémarrez ensuite le serveur.
